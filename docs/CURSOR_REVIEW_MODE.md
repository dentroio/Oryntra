# Easy review — no npm in the terminal

## One-time setup

1. `npm run build` (once after clone or big changes)
2. **Cursor Settings → MCP** — enable **oryntra** (from [`.cursor/mcp.json`](../.cursor/mcp.json))
3. Say **“Start Oryntra review”** in **Agent** mode — done.

## Optional: custom mode in dropdown

Only if you want a separate mode in the dropdown. Cursor’s UI varies — there may be **no “Features → Chat”** menu. See [`.cursor/oryntra-review-mode-instructions.md`](../.cursor/oryntra-review-mode-instructions.md): try **Settings → Chat** (speech-bubble icon) or search **custom modes**.

## Fallback

- Task: **Oryntra: Collaborate Now**
- `npm run collaborate`
